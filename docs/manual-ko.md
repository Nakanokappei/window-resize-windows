## 시작하기

1. **WindowResize.exe**를 실행합니다. 스플래시 화면이 잠시 표시됩니다.
2. 앱 아이콘이 **시스템 트레이**(작업 표시줄 오른쪽 하단의 알림 영역)에 나타납니다.
3. 트레이 아이콘을 클릭하여 메뉴를 엽니다.

> **참고:** 특별한 권한이 필요하지 않습니다. 실행 후 바로 사용할 수 있습니다.

## 창 크기 조절

### 사용 방법

1. 시스템 트레이에서 **Window Resize 아이콘**을 클릭합니다.
2. **"크기 조절"** 위에 마우스를 올려 창 목록을 엽니다.
3. 현재 열려 있는 모든 창이 **앱 아이콘**과 함께 **[앱 이름] 창 제목** 형식으로 표시됩니다. 긴 제목은 자동으로 잘립니다.
4. 창 위에 마우스를 올려 사용 가능한 프리셋 크기를 확인합니다.
5. 크기를 클릭하면 창이 즉시 조절됩니다.

### 크기 표시 형식

메뉴의 각 크기 항목은 다음과 같이 표시됩니다:

```
1920 x 1080          Full HD
```

- **왼쪽:** 너비 x 높이 (픽셀)
- **오른쪽:** 라벨 (표준 이름), 회색으로 표시

### 화면을 초과하는 프리셋 크기

프리셋 크기가 창이 위치한 디스플레이의 해상도를 초과하면 해당 크기는 **회색으로 표시되어 선택할 수 없습니다**. 이를 통해 창이 화면 경계를 넘어 크기가 조절되는 것을 방지합니다.

## 설정

Window Resize 트레이 아이콘을 클릭한 다음 **"설정..."**을 선택하여 설정 창을 엽니다.

### 기본 제공 크기

앱에는 12가지 기본 프리셋 크기가 포함되어 있습니다:

| 크기 | 라벨 |
|------|------|
| 3840 x 2160 | 4K UHD |
| 2560 x 1440 | QHD |
| 1920 x 1200 | WUXGA |
| 1920 x 1080 | Full HD |
| 1680 x 1050 | WSXGA+ |
| 1600 x 900 | HD+ |
| 1440 x 900 | WXGA+ |
| 1366 x 768 | WXGA |
| 1280 x 1024 | SXGA |
| 1280 x 720 | HD |
| 1024 x 768 | XGA |
| 800 x 600 | SVGA |

기본 제공 크기는 삭제하거나 편집할 수 없습니다.

### 사용자 정의 크기

자신만의 크기를 추가할 수 있습니다:

1. **너비**와 **높이**를 픽셀 단위로 입력합니다.
2. **"추가"**를 클릭합니다.
3. 새 크기가 사용자 정의 목록에 추가되고 크기 조절 메뉴에서 바로 사용할 수 있습니다.

사용자 정의 크기를 삭제하려면 옆에 있는 **"삭제"** 버튼을 클릭합니다.

### 로그인 시 시작

**"로그인 시 시작"**을 켜면 Windows에 로그인할 때 Window Resize가 자동으로 시작됩니다.

### 스크린샷

**"크기 조절 후 스크린샷 촬영"**을 켜면 창 크기 조절 완료 후 자동으로 창을 캡처합니다.

활성화하면 다음 옵션을 사용할 수 있습니다:

- **파일로 저장** — 스크린샷을 선택한 폴더에 PNG 파일로 저장합니다.
  > **파일명 형식:** `MMddHHmmss_앱이름_창제목.png` (예: `0227193012_chrome_Google.png`). 기호는 제거되고 문자, 숫자, 밑줄만 사용됩니다.
- **클립보드에 복사** — 스크린샷을 클립보드에 복사하여 다른 앱에 붙여넣을 수 있습니다.

두 옵션은 독립적으로 활성화할 수 있습니다.

## 문제 해결

### 크기 조절 실패

"크기 조절 실패" 알림이 표시되면 가능한 원인은 다음과 같습니다:

- 대상 창이 외부 크기 조절을 지원하지 않습니다.
- 창이 **전체 화면 모드**입니다 (**F11** 또는 **Esc**를 눌러 먼저 전체 화면을 종료하세요).

### 창이 목록에 표시되지 않음

크기 조절 메뉴에는 다음 조건을 충족하는 창만 표시됩니다:

- 현재 화면에 표시되어 있음
- 제목 표시줄이 있음
- Window Resize 앱 자체의 창이 아님

최소화된 창은 목록에 표시되지 않습니다.

### 스크린샷이 작동하지 않음

스크린샷이 캡처되지 않는 경우:

- 설정에서 **"파일로 저장"** 또는 **"클립보드에 복사"** 중 하나 이상이 활성화되어 있는지 확인합니다.
- 파일로 저장하는 경우 저장 폴더가 존재하고 쓰기 가능한지 확인합니다.
