## เริ่มต้นใช้งาน

1. เรียกใช้ **WindowResize.exe** หน้าจอเริ่มต้นจะแสดงสั้นๆ
2. ไอคอนแอปจะปรากฏใน **ซิสเต็มเทรย์** (พื้นที่แจ้งเตือนที่มุมขวาล่างของแถบงาน)
3. คลิกไอคอนเทรย์เพื่อเปิดเมนู

> **หมายเหตุ:** ไม่ต้องการสิทธิ์พิเศษใดๆ ใช้งานได้ทันทีหลังเปิดแอป

## ปรับขนาดหน้าต่าง

### ขั้นตอน

1. คลิก **ไอคอน Window Resize** ในซิสเต็มเทรย์
2. วางเมาส์เหนือ **"ปรับขนาด"** เพื่อเปิดรายการหน้าต่าง
3. หน้าต่างที่เปิดอยู่ทั้งหมดจะแสดงพร้อม **ไอคอนแอป** และชื่อในรูปแบบ **[ชื่อแอป] ชื่อหน้าต่าง** ชื่อที่ยาวจะถูกตัดให้สั้นลงอัตโนมัติ
4. วางเมาส์เหนือหน้าต่างเพื่อดูขนาดพรีเซ็ตที่มี
5. คลิกขนาดเพื่อปรับขนาดหน้าต่างทันที

### รูปแบบการแสดงขนาด

แต่ละรายการขนาดในเมนูแสดงดังนี้:

```
1920 x 1080          Full HD
```

- **ซ้าย:** กว้าง x สูง (พิกเซล)
- **ขวา:** ป้ายกำกับ (ชื่อมาตรฐาน) แสดงเป็นสีเทา

### ขนาดพรีเซ็ตที่เกินหน้าจอ

หากขนาดพรีเซ็ตใหญ่กว่าความละเอียดของจอแสดงผลที่หน้าต่างอยู่ ขนาดนั้นจะ **เป็นสีเทาและไม่สามารถเลือกได้** เพื่อป้องกันการปรับขนาดหน้าต่างเกินขอบเขตหน้าจอ

## การตั้งค่า

คลิกไอคอน Window Resize ในเทรย์ จากนั้นเลือก **"การตั้งค่า..."** เพื่อเปิดหน้าต่างการตั้งค่า

### ขนาดในตัว

แอปมีขนาดพรีเซ็ตในตัว 12 ขนาด:

| ขนาด | ป้ายกำกับ |
|------|-----------|
| 3840 x 2160 | 4K UHD |
| 2560 x 1440 | QHD |
| 1920 x 1200 | WUXGA |
| 1920 x 1080 | Full HD |
| 1680 x 1050 | WSXGA+ |
| 1600 x 900 | HD+ |
| 1440 x 900 | WXGA+ |
| 1366 x 768 | WXGA |
| 1280 x 1024 | SXGA |
| 1280 x 720 | HD |
| 1024 x 768 | XGA |
| 800 x 600 | SVGA |

ขนาดในตัวไม่สามารถลบหรือแก้ไขได้

### ขนาดที่กำหนดเอง

คุณสามารถเพิ่มขนาดของคุณเอง:

1. ป้อน **ความกว้าง** และ **ความสูง** เป็นพิกเซล
2. คลิก **"เพิ่ม"**
3. ขนาดใหม่จะเพิ่มในรายการและพร้อมใช้งานในเมนูปรับขนาดทันที

หากต้องการลบขนาดที่กำหนดเอง ให้คลิกปุ่ม **"ลบ"** ข้างๆ

### เริ่มต้นเมื่อเข้าสู่ระบบ

เปิด **"เริ่มต้นเมื่อเข้าสู่ระบบ"** เพื่อให้ Window Resize เริ่มต้นอัตโนมัติเมื่อคุณเข้าสู่ระบบ Windows

### ภาพหน้าจอ

เปิด **"ถ่ายภาพหน้าจอหลังปรับขนาด"** เพื่อจับภาพหน้าต่างอัตโนมัติหลังการปรับขนาด

เมื่อเปิดใช้งาน ตัวเลือกต่อไปนี้จะพร้อมใช้งาน:

- **บันทึกเป็นไฟล์** — บันทึกภาพหน้าจอเป็นไฟล์ PNG ไปยังโฟลเดอร์ที่คุณเลือก
  > **รูปแบบชื่อไฟล์:** `MMddHHmmss_ชื่อแอป_ชื่อหน้าต่าง.png` (เช่น `0227193012_chrome_Google.png`) สัญลักษณ์จะถูกลบออก ใช้เฉพาะตัวอักษร ตัวเลข และขีดล่าง
- **คัดลอกไปยังคลิปบอร์ด** — คัดลอกภาพหน้าจอไปยังคลิปบอร์ดเพื่อวางในแอปอื่น

ทั้งสองตัวเลือกสามารถเปิดใช้งานแยกกันได้

## การแก้ไขปัญหา

### ปรับขนาดล้มเหลว

หากแสดงข้อความ "ปรับขนาดล้มเหลว" สาเหตุที่เป็นไปได้:

- หน้าต่างเป้าหมายไม่รองรับการปรับขนาดจากภายนอก
- หน้าต่างอยู่ใน **โหมดเต็มหน้าจอ** (กด **F11** หรือ **Esc** เพื่อออกก่อน)

### หน้าต่างไม่ปรากฏในรายการ

เมนูปรับขนาดจะแสดงเฉพาะหน้าต่างที่:

- มองเห็นได้บนหน้าจอในปัจจุบัน
- มีแถบชื่อเรื่อง
- ไม่ใช่หน้าต่างของแอป Window Resize เอง

หน้าต่างที่ย่อเล็กจะไม่ปรากฏในรายการ

### ภาพหน้าจอไม่ทำงาน

หากภาพหน้าจอไม่ถูกจับภาพ:

- ตรวจสอบว่าเปิดใช้งาน **"บันทึกเป็นไฟล์"** หรือ **"คัดลอกไปยังคลิปบอร์ด"** อย่างน้อยหนึ่งอย่างในการตั้งค่า
- หากบันทึกเป็นไฟล์ ให้ตรวจสอบว่าโฟลเดอร์บันทึกมีอยู่และเขียนได้
