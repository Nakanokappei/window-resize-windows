## 开始使用

1. 运行 **WindowResize.exe**。启动画面会短暂显示。
2. 应用图标出现在 **系统托盘**（任务栏右下角的通知区域）。
3. 点击托盘图标打开菜单。

> **注意：** 无需特殊权限。启动后即可使用。

## 调整窗口大小

### 操作步骤

1. 点击系统托盘中的 **Window Resize 图标**。
2. 将鼠标悬停在 **"调整大小"** 上，打开窗口列表。
3. 所有当前打开的窗口都会显示其 **应用图标** 和名称，格式为 **[应用名] 窗口标题**。过长的标题会自动截断。
4. 将鼠标悬停在某个窗口上，查看可用的预设大小。
5. 点击某个大小，窗口将立即调整。

### 大小显示格式

菜单中的每个大小条目显示如下：

```
1920 x 1080          Full HD
```

- **左侧：** 宽 x 高（像素）
- **右侧：** 标签（标准名称），以灰色显示

### 超出屏幕的预设大小

如果预设大小超过窗口所在显示器的分辨率，该大小将 **变灰且无法选择**。这可以防止窗口超出屏幕边界。

## 设置

点击 Window Resize 托盘图标，然后选择 **"设置..."** 打开设置窗口。

### 内置大小

应用包含 12 种内置预设大小：

| 大小 | 标签 |
|------|------|
| 3840 x 2160 | 4K UHD |
| 2560 x 1440 | QHD |
| 1920 x 1200 | WUXGA |
| 1920 x 1080 | Full HD |
| 1680 x 1050 | WSXGA+ |
| 1600 x 900 | HD+ |
| 1440 x 900 | WXGA+ |
| 1366 x 768 | WXGA |
| 1280 x 1024 | SXGA |
| 1280 x 720 | HD |
| 1024 x 768 | XGA |
| 800 x 600 | SVGA |

内置大小无法删除或编辑。

### 自定义大小

您可以添加自己的大小：

1. 输入 **宽度** 和 **高度**（像素）。
2. 点击 **"添加"**。
3. 新大小将添加到自定义列表中，并立即在调整大小菜单中可用。

要删除自定义大小，请点击旁边的 **"删除"** 按钮。

### 登录时启动

开启 **"登录时启动"** 后，Window Resize 将在您登录 Windows 时自动启动。

### 截图

开启 **"调整大小后截图"** 后，调整窗口大小完成后将自动捕获窗口截图。

启用后，以下选项可用：

- **保存到文件** — 将截图保存为 PNG 文件到您选择的文件夹。
  > **文件名格式：** `MMddHHmmss_应用名_窗口标题.png`（例如：`0227193012_chrome_Google.png`）。符号会被去除，仅使用字母、数字和下划线。
- **复制到剪贴板** — 将截图复制到剪贴板，以便粘贴到其他应用中。

两个选项可以独立启用。

## 故障排除

### 调整大小失败

如果出现"调整大小失败"的提示，可能的原因包括：

- 目标窗口不支持外部调整大小。
- 窗口处于 **全屏模式**（请先按 **F11** 或 **Esc** 退出全屏）。

### 窗口未出现在列表中

调整大小菜单仅显示满足以下条件的窗口：

- 当前在屏幕上可见
- 具有标题栏
- 不是 Window Resize 应用自身的窗口

最小化的窗口不会出现在列表中。

### 截图未生效

如果截图未被捕获：

- 确保在设置中启用了 **"保存到文件"** 或 **"复制到剪贴板"** 中的至少一个。
- 如果保存到文件，请确认保存文件夹存在且可写。
