## 開始使用

1. 執行 **WindowResize.exe**。啟動畫面會短暫顯示。
2. 應用程式圖示出現在 **系統匣**（工作列右下角的通知區域）。
3. 點按系統匣圖示開啟選單。

> **注意：** 無需特殊權限。啟動後即可使用。

---

## 調整視窗大小

### 操作步驟

1. 點按系統匣中的 **Window Resize 圖示**。
2. 將滑鼠移至 **「調整大小」** 上，開啟視窗列表。
3. 所有目前開啟的視窗都會顯示其 **應用程式圖示** 和名稱，格式為 **[應用程式名稱] 視窗標題**。過長的標題會自動截斷。
4. 將滑鼠移至某個視窗上，檢視可用的預設大小。
5. 點按某個大小，視窗將立即調整。

### 大小顯示格式

選單中的每個大小項目顯示如下：

```
1920 x 1080          Full HD
```

- **左側：** 寬 x 高（像素）
- **右側：** 標籤（標準名稱），以灰色顯示

### 超出螢幕的預設大小

如果預設大小超過視窗所在顯示器的解析度，該大小將 **變灰且無法選取**。這可以防止視窗超出螢幕邊界。

## 設定

點按 Window Resize 系統匣圖示，然後選取 **「設定...」** 開啟設定視窗。

### 內建大小

應用程式包含 12 種內建預設大小：

| 大小 | 標籤 |
|------|------|
| 3840 x 2160 | 4K UHD |
| 2560 x 1440 | QHD |
| 1920 x 1200 | WUXGA |
| 1920 x 1080 | Full HD |
| 1680 x 1050 | WSXGA+ |
| 1600 x 900 | HD+ |
| 1440 x 900 | WXGA+ |
| 1366 x 768 | WXGA |
| 1280 x 1024 | SXGA |
| 1280 x 720 | HD |
| 1024 x 768 | XGA |
| 800 x 600 | SVGA |

內建大小無法刪除或編輯。

### 自訂大小

您可以新增自己的大小：

1. 輸入 **寬度** 和 **高度**（像素）。
2. 點按 **「新增」**。
3. 新大小將新增到自訂列表中，並立即在調整大小選單中可用。

要移除自訂大小，請點按旁邊的 **「移除」** 按鈕。

### 登入時啟動

開啟 **「登入時啟動」** 後，Window Resize 將在您登入 Windows 時自動啟動。

### 螢幕截圖

開啟 **「調整大小後擷取螢幕截圖」** 後，調整視窗大小完成後將自動擷取視窗螢幕截圖。

啟用後，以下選項可用：

- **儲存至檔案** — 將螢幕截圖儲存為 PNG 檔案到您選擇的資料夾。
  > **檔案名稱格式：** `MMddHHmmss_應用程式名稱_視窗標題.png`（例如：`0227193012_chrome_Google.png`）。符號會被移除，僅使用字母、數字和底線。
- **複製到剪貼簿** — 將螢幕截圖複製到剪貼簿，以便貼上到其他應用程式中。

兩個選項可以獨立啟用。

## 疑難排解

### 調整大小失敗

如果出現「調整大小失敗」的提示，可能的原因包括：

- 目標視窗不支援外部調整大小。
- 視窗處於 **全螢幕模式**（請先按 **F11** 或 **Esc** 退出全螢幕）。

### 視窗未出現在列表中

調整大小選單僅顯示滿足以下條件的視窗：

- 目前在螢幕上可見
- 具有標題列
- 不是 Window Resize 應用程式自身的視窗

最小化的視窗不會出現在列表中。

### 螢幕截圖未生效

如果螢幕截圖未被擷取：

- 確保在設定中啟用了 **「儲存至檔案」** 或 **「複製到剪貼簿」** 中的至少一個。
- 如果儲存至檔案，請確認儲存資料夾存在且可寫入。
