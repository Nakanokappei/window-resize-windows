## Начало работы

1. Запустите **WindowResize.exe**. Кратковременно отобразится заставка.
2. Значок приложения появится в **области уведомлений** (в правом нижнем углу панели задач).
3. Нажмите на значок в трее, чтобы открыть меню.

> **Примечание:** Специальные разрешения не требуются. Приложение работает сразу после запуска.

## Изменение размера окна

### Пошаговая инструкция

1. Нажмите на **значок Window Resize** в области уведомлений.
2. Наведите курсор на **«Изменить размер»**, чтобы открыть список окон.
3. Все открытые окна отображаются со своим **значком приложения** и названием в формате **[Имя приложения] Заголовок окна**. Длинные заголовки автоматически сокращаются.
4. Наведите курсор на окно, чтобы увидеть доступные предустановленные размеры.
5. Нажмите на размер, чтобы немедленно изменить размер окна.

### Формат отображения размеров

Каждая запись размера в меню отображается так:

```
1920 x 1080          Full HD
```

- **Слева:** Ширина x Высота (в пикселях)
- **Справа:** Метка (стандартное название), отображается серым

### Предустановки, превышающие размер экрана

Если предустановленный размер больше разрешения экрана, на котором находится окно, этот размер будет **неактивным и недоступным для выбора**.

## Настройки

Нажмите на значок Window Resize в трее, затем выберите **«Настройки...»**, чтобы открыть окно настроек.

### Встроенные размеры

Приложение включает 12 встроенных предустановленных размеров:

| Размер | Метка |
|--------|-------|
| 3840 x 2160 | 4K UHD |
| 2560 x 1440 | QHD |
| 1920 x 1200 | WUXGA |
| 1920 x 1080 | Full HD |
| 1680 x 1050 | WSXGA+ |
| 1600 x 900 | HD+ |
| 1440 x 900 | WXGA+ |
| 1366 x 768 | WXGA |
| 1280 x 1024 | SXGA |
| 1280 x 720 | HD |
| 1024 x 768 | XGA |
| 800 x 600 | SVGA |

Встроенные размеры нельзя удалить или изменить.

### Пользовательские размеры

Вы можете добавить собственные размеры:

1. Введите **Ширину** и **Высоту** в пикселях.
2. Нажмите **«Добавить»**.
3. Новый размер добавляется в список и сразу становится доступным в меню изменения размера.

Чтобы удалить пользовательский размер, нажмите кнопку **«Удалить»** рядом с ним.

### Запуск при входе в систему

Включите **«Запуск при входе в систему»**, чтобы Window Resize автоматически запускался при входе в Windows.

### Снимок экрана

Включите **«Сделать снимок экрана после изменения размера»**, чтобы автоматически захватывать окно после изменения размера.

При включении доступны следующие параметры:

- **Сохранить в файл** — Сохранить снимок экрана как PNG-файл в выбранную папку.
  > **Формат имени файла:** `MMddHHmmss_ИмяПриложения_ЗаголовокОкна.png` (например: `0227193012_chrome_Google.png`). Символы удаляются.
- **Копировать в буфер обмена** — Копировать снимок экрана в буфер обмена для вставки в другие приложения.

Оба параметра можно включить независимо друг от друга.

## Устранение неполадок

### Не удалось изменить размер

Если появляется сообщение «Не удалось изменить размер»:

- Целевое окно не поддерживает внешнее изменение размера.
- Окно находится в **полноэкранном режиме** (нажмите **F11** или **Esc** для выхода).

### Окно не отображается в списке

Меню изменения размера показывает только окна, которые:

- В данный момент видимы на экране
- Имеют строку заголовка
- Не являются окнами самого приложения Window Resize

Свёрнутые окна не отображаются в списке.

### Снимок экрана не работает

Если снимки экрана не выполняются:

- Убедитесь, что в настройках включен хотя бы один из параметров **«Сохранить в файл»** или **«Копировать в буфер обмена»**.
- При сохранении в файл убедитесь, что папка назначения существует и доступна для записи.
